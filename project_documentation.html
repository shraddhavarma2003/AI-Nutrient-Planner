üìã Project Overview

Project Name: Context-Aware Personalized Nutrition Planner

Description: A healthcare-support system (NOT a medical diagnosis tool) that helps users log meals, track nutrition, and receive personalized guidance based on their health conditions.

Key Features:

  • Multi-modal meal logging (text, image, voice)
  • RAG-powered nutrition lookup from USDA/OpenFoodFacts
  • Medical Safety Engine (allergies, diabetes, hypertension, obesity)
  • Smart meal corrections ("Fix My Meal")
  • Reverse Recipe Generation
  • Context-aware Virtual Coach

  6. Data Layer (src/models/, data/): Database and data models
  7. External Services (src/services/): LLM APIs (Mistral)
